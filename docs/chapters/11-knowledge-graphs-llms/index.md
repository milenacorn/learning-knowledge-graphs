# Knowledge Graphs and Large Language Models

## Summary

This chapter explores how GraphRAG and retrieval-augmented generation use
knowledge graphs to ground large language model outputs and reduce
hallucination, covering vector databases, semantic and hybrid retrieval, and
LLM-assisted extraction. It also covers the practical mechanics of context
windows, chunking strategies, and fact verification. Readers will finish
able to explain how a knowledge graph improves the factual reliability of an
LLM-based system.
