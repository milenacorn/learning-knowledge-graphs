# Knowledge Graphs and Machine Learning

## Summary

This chapter explains how knowledge graph embeddings and graph neural
networks turn graph-structured data into a form machine-learning models can
consume, covering node embeddings, TransE, message passing, graph
convolution, and random-walk methods like Node2Vec. It then applies these
representations to link prediction, node classification, and knowledge
graph completion. Readers will finish able to explain how an embedding
model turns nodes and edges into vectors, and what those vectors are used for.
