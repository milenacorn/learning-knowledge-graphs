# Data Storage Paradigms and Graph Variants

## Summary

This chapter contrasts graph databases with the relational tables and document
stores most technical readers already know, making explicit what graph
storage does differently. It then surveys graph variants — multigraphs,
hypergraphs, weighted graphs, and the cyclic/acyclic distinction — that
recur throughout the rest of the book. Readers will finish able to explain
why a graph database is a different tool than a relational database, not
just a different syntax for the same one.
