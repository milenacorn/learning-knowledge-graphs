# Graph Data Models

## Summary

This chapter compares RDF triples and labeled property graphs as the two
dominant ways to represent a knowledge graph, covering blank nodes, named
graphs, reification, and the tradeoffs between triple stores and quad
stores. It culminates in a formal definition of "knowledge graph" itself —
the central concept this entire book is organized around. Readers will
finish able to choose between the RDF and property-graph models for a
given project and justify that choice.

## Concepts Covered

This chapter covers the following 11 concepts from the learning graph:

1. RDF Triple
2. Property Graph
3. Labeled Property Graph
4. Blank Node
5. Named Graph
6. Reification
7. Quad Store
8. Triple Store
9. Graph Model Tradeoffs
10. Directed Acyclic Graph
11. Knowledge Graph
