# Building Knowledge Graphs from Data

## Summary

This chapter walks through the ETL pipeline used to construct a knowledge
graph from real-world sources: data ingestion, entity extraction, named
entity recognition, relation extraction, and entity resolution, followed by
deduplication, record linkage, and data cleaning. It closes with knowledge
extraction and text mining as the techniques that turn unstructured text
into graph-ready triples. Readers will finish able to take a raw data
source and produce a populated, de-duplicated knowledge graph from it.
