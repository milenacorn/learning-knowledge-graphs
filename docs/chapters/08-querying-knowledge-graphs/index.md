# Querying Knowledge Graphs

## Summary

This chapter teaches SPARQL and Cypher query patterns — SELECT, CONSTRUCT,
and ASK queries, triple patterns, and MATCH clauses — plus the Python client
libraries (RDFLib, Neo4j drivers) used to run them against a live knowledge
graph. It also covers query optimization and federated queries across
multiple endpoints. Readers will finish able to retrieve and traverse data
from either an RDF triple store or a property graph database.
