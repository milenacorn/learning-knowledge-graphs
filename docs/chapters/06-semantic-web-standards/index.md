# Semantic Web Standards

## Summary

This chapter introduces RDF, RDFS, OWL, SKOS, and SPARQL — the standards
that make knowledge graphs interoperable across systems and organizations —
alongside the URIs, namespaces, and serialization formats (Turtle, JSON-LD,
N-Triples) used to write them down. It also covers the OWL class and
property constructs (equivalence, disjointness, transitivity, symmetry,
inverse, cardinality restrictions) used to formalize a schema precisely.
Readers will finish able to read and write basic RDF/OWL and explain what
each standard is for.
